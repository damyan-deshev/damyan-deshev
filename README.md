# Damyan Deshev

I build local-first AI systems that make model behavior inspectable: exact recall, evidence logs, token-level diagnostics, benchmark harnesses, and deterministic product workflows.

Most of my work sits at the edge between infrastructure, retrieval, model evaluation, and small local tools that solve real problems without turning into platforms for their own sake.

Selected work:

- `ariadne` - full workbench: long chats, recall, personas, evidence workflows, TTS, and llama.cpp debugging.
- `logit_snake` - inspection lab: token probabilities, decoder risk, branching, embeddings, and consistency checks.
- `vera` - research runtime: sandboxed tools, evidence logs, verification loops, and local agent workflows.
- `damyan-bench` - evaluation harness: OpenAI-compatible coding, tool-use, boundary, and operator-style benchmarks.
- `myroll` - product system: private GM prep, explicit public publishing, maps, fog, tokens, and player-facing scenes.
- `tavichka` - deterministic search: mobile recipe search with USDA nutrition mapping and in-memory indexes.
